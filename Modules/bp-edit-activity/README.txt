Allow users to edit their BuddyPress activity posts and comments from the front-end.

== Description ==

BuddyPress Edit Activity enhances your BuddyPress community by allowing members to edit their activity posts and comments directly from the front end of your website. This plugin provides a clean, intuitive interface for making corrections or updates to shared content.

With BuddyPress Edit Activity, users can maintain accuracy in their posts without cluttering the activity stream with correction comments or duplicate posts. Administrators can configure time limits for editing to ensure content integrity while giving users flexibility to fix errors.

= Key Features =

* **Front-End Editing**: Users can edit their activity posts directly from the activity stream
* **Time-Based Restrictions**: Set time limits for how long users can edit their posts (Forever, 30 Days, 7 Days, 1 Day, 1 Hour, or just 10 Minutes)
* **Edit Indicator**: Posts that have been edited display an "(edited)" indicator for transparency
* **Multiple Locations**: Edit functionality works in sitewide activity streams, group pages, and user profiles
* **User-Friendly Interface**: Clean modal popup for editing with familiar controls
* **RTL Support**: Fully compatible with right-to-left languages
* **Developer Friendly**: Includes hooks and filters for customization

= BuddyPress Edit Activity is perfect for: =

* Communities that value content accuracy and quality
* Forums where discussions evolve and users need to update their contributions
* Educational platforms where students may need to correct submissions
* Professional networks where accuracy of information is critical

= Compatibility =

* WordPress 4.0+
* BuddyPress 3.0+
* Compatible with BuddyPress Giphy, BuddyPress Check-ins, and more
* Works with most BuddyPress-compatible themes
* Compatible with Youzify (formerly YouzerBP)

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'BuddyPress Edit Activity'
3. Activate BuddyPress Edit Activity from your Plugins page
4. Visit 'Settings > BuddyPress > Edit Activity' to configure edit duration settings

= From WordPress.org =

1. Download BuddyPress Edit Activity
2. Upload the 'buddypress-edit-activity' directory to your '/wp-content/plugins/' directory
3. Activate BuddyPress Edit Activity from your Plugins page
4. Visit 'Settings > BuddyPress > Edit Activity' to configure edit duration settings

== Frequently Asked Questions ==

= What is BuddyPress Edit Activity? =

BuddyPress Edit Activity is a plugin that allows users to edit their activity posts and comments directly from the frontend of your BuddyPress-powered site. It helps maintain content accuracy and improves user engagement.

= Can users edit their activity posts sitewide? =

Yes, users can edit their posts in the Sitewide Activity Stream, Single Group Pages, and User Activity Pages, ensuring their shared content remains relevant and up-to-date.

= How long can users edit their activity posts? =

Administrators can configure the editing duration with options such as Forever, 30 Days, 7 Days, 1 Day, 1 Hour, or 10 minutes, allowing flexible control over post modifications.

= Can users edit activity comments? =

Yes, users can edit their own comments on activity posts, just like they can edit the posts themselves, ensuring accuracy in conversations and community discussions.

= Does the plugin work with BuddyPress Groups? =

Yes, BuddyPress Edit Activity allows users to edit their posts within BuddyPress Groups, enabling better collaboration and accurate discussions within group activity feeds.

= Does this plugin support other BuddyPress add-ons? =

Yes, BuddyPress Edit Activity seamlessly integrates with BuddyPress Check-ins, BuddyPress Quotes, BuddyPress Status, BuddyPress Giphy, and BuddyPress Polls, allowing users to modify check-ins, quotes, status updates, GIFs, and poll details after posting.

= How do users edit an activity post? =

Users can locate their post in the activity stream or user profile, click the Edit button next to the post, make the necessary changes, and click Update Activity to save the edits.
